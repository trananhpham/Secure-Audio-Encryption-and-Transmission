# Sample Data

Đây là thư mục chứa các file âm thanh phục vụ cho việc kiểm thử và chạy dự án.
Để tránh vi phạm bản quyền và giảm thiểu dung lượng khi đẩy lên GitHub, các file âm thanh gốc sẽ không được upload.

## Hướng dẫn

1. Tạo thư mục `mp3/` và `wav/` tại đây nếu chưa có.
2. Sao chép và đổi tên các file âm thanh của bạn thành `at1.mp3`, `at2.mp3`, `at3.mp3`, `at4.mp3`, `at5.mp3` và đặt vào thư mục `mp3/`.
3. Tương tự cho định dạng `.wav`.
